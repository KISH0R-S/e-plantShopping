# Paradise Nursery Shopping Application

Paradise Nursery is an online plant shop that allows users to browse houseplants, add them to a shopping cart, and manage their cart items.

## Features
- Browse plants organized into 3 categories (Air Purifying, Tropical, Succulents & Cacti)
- Add plants to cart (button disables after adding)
- Dynamic cart count in navbar
- Manage cart: increase/decrease quantity, delete items
- Total cost calculation per item and overall
- Checkout "Coming Soon" message
- Continue Shopping button

## Technologies
- React 18
- Redux Toolkit
- React Router DOM v6
- Vite

## Getting Started

```bash
npm install
npm run dev
```
